Using System Management Database

Overview

The Systems Management Database is a Lotus Notes Database for flagging requests or issues encountered which requires the support of the Systems Solutions Team except for STC eTools

Purpose of this Database:
a. Originators can raise Request Tickets in this Database.
b. This allows Systems Solutions Contact (SSC) and where required, Advisors to act on and resolve these Request Tickets.

Getting Started

There are two main sections in the database, "Request Tickets" & "Administration".
The Request Tickets section contains all information regarding the Requests. The Users will be able to use this section. This is where they can submit and keep track of their submitted Request Tickets.
The Administration section contains all information regarding the documents used for set up. Only the Administrator has access to information in this section.
